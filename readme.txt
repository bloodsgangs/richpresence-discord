1. Vous devez d’abord enregistrer une application Rich Presence avec discord
- Allez ici https://discordapp.com/developers/applications/me
- Faire une nouvelle application **Le nom de l’application sera le nom principal de la présence riche**
- Activer la présence enrichie pour votre application et ajouter des ressources
2 Téléchargez la dernière version de naco à partir d’ici https://github.com/bloodsgangs/richpresence-discord
4 Exécutez naco (il devrait ouvrir une fenêtre cmd)
    - Il * devrait * signaler les erreurs de votre fichier de configuration (s’il y en a)
5 Discord devrait afficher le jeu sur votre profil
	- sinon, ajoutez l’exe en tant que jeu sur discord et le chemin du fichier devrait changer en votre présence

Vous pouvez modifier la configuration à tout moment pendant l’exécution du programme pour modifier la présence (assurez-vous d’enregistrer le fichier)
